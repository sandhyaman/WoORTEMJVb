# 王者RY战队数据分析系统 python1175

## 项目概述

本项目是基于Python开发的王者RY战队数据分析系统，主要用于分析战队相关数据，提升决策效率。

## 技术栈

- 后端：Python
- 数据库：MYSQL

## 功能模块

### 一、首页

- 显示战队数据概览

### 二、比赛信息管理

- 比赛数据的增删改查
- 比赛数据分析

### 三、系统管理

- 用户管理
- 权限管理

### 四、我的资料

- 修改个人资料
- 查看个人数据

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- MYSQL数据库

## 部署步骤

1. 安装Python环境
2. 安装MYSQL数据库
3. 导入数据库结构及数据
4. 部署后端代码
5. 启动项目

## 目录结构

```
项目目录/
├── backend                # 后端代码目录
│   ├── app                # Flask应用目录
│   ├── models             # 数据模型目录
│   └── views              # 视图目录
├── database               # 数据库相关目录
│   ├── schema.sql         # 数据库结构文件
│   └── data.sql           # 数据库数据文件
└── README.md              # 项目说明文件
```
